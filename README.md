This project was completed as part of the Cognifyz Technologies Web Development Internship Program. The project includes Level 1 and Level 3 tasks such as creating a responsive webpage with HTML and CSS, developing interactive JavaScript functionalities including a color-changing button and calculator, designing an image gallery, and building a professional internship landing page. The project demonstrates front-end development skills, responsive design principles, user interface design, and JavaScript interactivity. The objective was to gain practical experience in web development and strengthen problem-solving and coding skills through real-world tasks.
